**In `src/storage/export.ts`:**

Add an exported `exportCorrectionLog()` async function that:
1. Dynamically imports `getClassificationHistory` from `./classification-log`
2. Filters events where `suggestedType !== undefined && suggestedType !== chosenType` (skip events with no suggestion — they can't be corrections; see Pitfall 1 in RESEARCH.md)
3. If no corrections found, return early (no download triggered)
4. Maps each correction to a JSON object with fields: `content`, `suggestedType`, `chosenType`, `tier` (default null), `confidence` (default null), `timestamp`
5. Joins with `\n` and adds trailing `\n` (Pitfall 4: Python JSONL parsers expect trailing newline)
6. Creates a Blob with MIME type `application/x-ndjson`
7. Calls `triggerDownload(blob, 'binderos-corrections-YYYY-MM-DD.jsonl')` using the existing private helper
8. Returns the correction count so the caller can update its local state

Change the return type to `Promise<number>` (returns count of corrections exported, or 0 if none).
