Create script 31_train_decomposition_classifier.py following 21_train_gtd_classifier.py pattern exactly.

Configuration:
- hidden_layers: (256, 128) — multi-class architecture matching Phase 17 pattern
- max_iter: 500
- Input: decomposition.jsonl
- Output: decomposition.onnx + decomposition-classes.json
- Embed all texts using sentence-transformers all-MiniLM-L6-v2 with normalize_embeddings=True
- Train/test split: 80/20 stratified
- MLPClassifier with relu activation, adam solver
- CalibratedClassifierCV(method='sigmoid') for Platt calibration
- Export via skl2onnx with opset 17
- Save test embeddings + Python predictions/probabilities for validation script
- Print accuracy, confusion matrix summary, per-class accuracy for worst 5 classes

- Report any prediction mismatches with their categories
